# Customer-Plan-Recommendation-Model-for-Megaline-Telecom
## Project Description

Megaline Telecom wants to migrate customers from legacy plans to new plans (Smart and Ultra).  
This project develops a machine learning classification model capable of recommending the best plan based on customer behavior.

## Dataset

The dataset contains behavioral information from mobile subscribers including:

- number of calls
- minutes used
- text messages sent
- mobile data usage
- current plan type

## Objective

Build a classification model that predicts whether a customer should use the **Smart** or **Ultra** plan.

Project requirement:
Accuracy ≥ 0.75

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models Tested

- Decision Tree
- Random Forest
- Logistic Regression

## Evaluation Metric

Accuracy

## Results

The best model achieved an accuracy above the required threshold of **0.75**, successfully predicting the most suitable plan for customers.
